💡 使用说明

模块化版本：推荐使用，代码结构清晰，易于维护和扩展。需要本地服务器环境。

单文件版本：可以直接在浏览器中打开使用，适合快速预览和演示。

本地服务器：如果使用模块化版本，建议使用本地服务器（如 Python 的 http.server 或 VS Code 的 Live Server）来避免跨域问题。
