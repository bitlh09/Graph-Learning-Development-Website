图的基本概念

现在我们正式学习图理论的基础知识。

图的定义

图 G = (V, E) 由以下组成：

  • • V: 节点集合 (Vertices)
  • • E: 边集合 (Edges)
  • • 每条边连接两个节点

常用术语

  • • 度数：节点的连接数
  • • 邻居：直接相连的节点
  • • 路径：节点间的连接序列
  • • 直径：最短路径长度

邻接矩阵表示

邻接矩阵 A 是表示图结构的数学方法：

  • • A[i][j] = 1 表示节点 i 和 j 之间有连接
  • • A[i][j] = 0 表示节点 i 和 j 之间无连接
  • • 对于无向图，邻接矩阵是对称的

神经网络回顾

在学习图神经网络之前，让我们先回顾一下传统神经网络的的基本概念。

全连接层 (FC)

  • • 每个输入连接所有输出
  • • 适用于表格数据
  • • 参数量大

卷积层 (CNN)

  • • 局部连接，共享参数
  • • 适用于图像数据
  • • 具有平移不变性

循环层 (RNN)

  • • 具有记忆能力
  • • 适用于序列数据
  • • 能处理变长输入

传统神经网络的局限性

  • • FC层：无法利用数据的空间结构
  • • CNN：只能处理欧几里得空间中的规则数据
  • • RNN：只能处理序列结构，不适用于复杂网络

为什么需要图神经网络？

现实世界中存在大量的图结构数据，传统神经网络无法直接处理。

图数据的特点

  • • 非欧几里得空间：节点没有固定的空间位置
  • • 变大小：不同图的节点数量不同
  • • 无固定顺序：节点没有自然的排列顺序
  • • 网络效应：节点的特性受邻居影响

GNN的优势

  • • 排列不变性：不依赖节点的输入顺序
  • • 局部性：每个节点只关注直接邻居
  • • 共享参数：所有节点使用相同的参数
  • • 归纳能力：可以处理未见过的图结构

消息传递机制

消息传递是图神经网络的核心思想：节点通过与邻居交换信息来更新自己的表示。

消息传递的三个步骤

  1. 1. 消息收集 (Message Aggregation)
    每个节点收集来自邻居节点的信息
  2. 2. 消息聚合 (Message Combining)
    将收集到的多个消息整合成一个表示
  3. 3. 状态更新 (State Update)
    结合自身信息和邻居信息，更新节点表示

数学表达

消息函数： mij(l) = Message(hi(l-1), hj(l-1), eij)

聚合函数： ai(l) = Aggregate({mji(l) : j ∈ N(i)})

更新函数： hi(l) = Update(hi(l-1), ai(l))

图的类型与表示

有向图

  • • 边有方向，表示非对称关系
  • • 例如：微博关注、网页链接

无向图

  • • 边无方向，表示对称关系
  • • 例如：友谊网络、分子结构

GNN通用框架

消息传递框架

步骤 1: 消息函数

m_{ij} = Message(h_i, h_j, e_{ij})

步骤 2: 聚合函数

m_i = Aggregate({m_{ij} : j ∈ N(i)})

步骤 3: 更新函数

h_i^{new} = Update(h_i^{old}, m_i)

图卷积网络 (GCN)

核心思想

  • • 将卷积操作从欧几里得空间扩展到图空间
  • • 通过聚合邻居节点信息更新节点表示
